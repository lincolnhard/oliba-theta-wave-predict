1.下載 MAMP 安裝檔並安裝
2.更改時區，打開 MAMP 查看 Preferences > PHP 裡面的版本號，到 /MAMP/bin/php/php[版本]/php.ini 裡面，搜尋 "date.timezone"，確認該行如下
    date.timezone = "Asia/Taipei"
3.Preferences > Web Server 將網站路徑指向專案資料夾
4.打開瀏覽器輸入 localhost （如果 Preferences > Ports 裡面的 Apache Port 不是 80，改輸入 localhost:[Apache Port]）
